mung# AULIASOFT Promotional Website

Static single-page promotional website for AULIASOFT POS, an Indonesian web-based point-of-sale product for retail stores, food businesses, small shops, and cooperatives.

## Stack

- HTML5
- CSS3 with custom properties
- Vanilla JavaScript
- GSAP + ScrollTrigger via CDN
- Lenis smooth scrolling via CDN
- Font Awesome via CDN
- Google Fonts Inter

## Main Files

- `index.html` - main single-page landing page
- `css/custom.css` - active stylesheet
- `js/custom.js` - active JavaScript
- `img/` - logo and product screenshots
- `robots.txt` - crawler rules
- `sitemap.xml` - sitemap entry
- `PRD.md` - product requirements document

## Features

- Responsive landing page
- Dark/light theme toggle with `localStorage`
- Smooth scrolling navigation
- Scroll-triggered animations
- Mobile drawer menu
- FAQ accordion
- WhatsApp contact form redirect
- Pricing cards and product showcase
- SEO meta tags, Open Graph, Twitter Card, and JSON-LD structured data

## Development

Open `index.html` directly in a browser or serve the directory using a local web server such as Laragon.

No package installation or build step is required.
